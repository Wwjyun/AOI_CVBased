NG Tile 面積分類小工具 v1.0.0
================================

用途
----
依 VisionFlow AOI 缺陷 CSV 的 area 欄位，自動分類 ng_tiles 圖片。

使用方式
--------
1. 執行「NG Tile 面積分類小工具.exe」。
2. 選擇同時包含 csv 與 ng_tiles 的 AOI 輸出根資料夾。
3. 每行輸入一個面積區間，例如：
   200-400
   401-500
4. 選擇同一張 Tile 有多筆缺陷時，要使用最大面積、面積總和或最小面積。
5. 按下「開始分類」。

輸出
----
預設輸出至所選根資料夾下的 area_classified 資料夾。
程式只會複製圖片，不會搬移或刪除原始 CSV、NG Tile 或 JSON。
未落入任何區間的圖片預設放入「_未落入區間」。

面積單位
--------
程式會讀取 CSV 的 area_unit。舊 CSV 若沒有 area_unit，會視為 px^2。
若同一根資料夾混有 px^2 與 um^2，輸出會自動分成 px2 與 um2，避免混用。
